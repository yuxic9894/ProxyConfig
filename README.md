🚀 Clash 极简高性能远程配置 (Sub-Converter 版)
这是一份经过深度优化的 Clash 订阅转换 .ini 配置文件。它不仅实现了精准分流、自动测速，还针对 2026 年主流 AI 工具（ChatGPT、Claude、Gemini）进行了专项加速与解锁优化。

✨ 配置亮点
🤖 AI 专项优化：新增 ✨ AI Tools 分组，集成 OpenAI、Claude 和 Gemini 规则，默认优先选择稳定性高的地区（新加坡/美国/日本）。

🌍 精准国家分组：通过正则表达式自动筛选 香港、台湾、美国、日本、韩国、新加坡，并设有独立的“其他节点”组（排除法匹配，自动收纳欧洲、南美等小众节点）。

⚡ 自动优选策略：

url-test：自动选择延迟最低节点，容差设为 150ms，有效防止因微小网络波动导致的频繁断流。

故障转移 (Fallback)：核心总控组支持故障自动切换。

🛡️ 强力广告过滤：多重规则嵌套（ACL4SSR + AdBlock），过滤网页广告及应用内垃圾信息。

🎮 游戏/视频加速：针对 Steam、Epic、Netflix、YouTube 及 Bilibili 港澳台提供独立分流开关。

🚀 如何使用
复制配置：复制 [custom] 以下的所有内容。

订阅转换：

打开任意订阅转换后端（如 ACL4SSR 在线转换）。

在“远程配置”或“自定义配置”处，贴入此配置。

建议勾选 UDP、TFO 以及 自动设置 Fake-IP。

导入 Clash：生成订阅链接并导入 Clash (Windows/Android/Mac) 即可开始使用。

🔍 正则匹配规则参考
如果你想修改“其他节点”的排除范围，请关注以下正则逻辑：

代码段
^(?!.*(港|HK|台|TW|美|US|日|JP|韩|KR|新|SG)).*
注：该正则会自动避开所有包含关键词的节点，将剩余节点（如英国 UK、德国 DE 等）聚合，适合喜欢“开盲盒”探索新线路的用户。
